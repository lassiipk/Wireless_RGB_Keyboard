# Wireless RGB Mechanical Keyboard (96%)

## About
I am building a custom mechanical keyboard with USB-C and Bluetooth.
The goal is to learn electronics, PCB design, and firmware, while
creating a keyboard I can fully control and modify.

## Features I Plan to Build
- USB-C wired mode
- Bluetooth wireless mode
- Per-key RGB lighting
- Large battery for long usage
